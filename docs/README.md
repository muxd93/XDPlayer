## 项目一句话简介

MzDKPlayer 是一款专为 Android TV 设计的本地音视频播放器，支持弹幕、多网络协议（SMB/FTP/WebDAV/NFS/HTTP）、TMDB 刮削，并具备适老化"老人模式"与 Web 远程配置能力，可作为 TV Launcher 使用。
