# 01 - 项目概述

## 1.1 项目背景

MzDKPlayer 是一款专为 **Android TV**（安卓电视盒子/智能电视）设计的本地音视频播放器。项目起源于个人对家庭影音中心的需求：电视端缺乏一款同时支持**局域网 NAS 多协议播放**、**B 站风格弹幕**、**TMDB 影视信息刮削**的免费开源播放器。

项目定位为"能跑就行"的实用工具，代码风格务实，优先功能可用性而非架构完美性。

## 1.2 项目目标

| 维度 | 目标 |
|------|------|
| **播放能力** | 支持本地与局域网（SMB/FTP/WebDAV/NFS/HTTP）视频/音频/图片播放 |
| **弹幕体验** | 支持 B 站风格 XML 弹幕加载与渲染 |
| **影视信息** | 通过 TMDB 自动获取电影/电视剧海报、简介、评分 |
| **适老化** | 提供"老人模式"，零文字依赖，家属可通过 Web 页面远程配置 |
| **Launcher** | 可作为 Android TV 默认桌面，开机直达 |
| **格式兼容** | 覆盖常见封装（MP4/MKV/ISO 蓝光原盘等）与音频直通 |

## 1.3 核心功能

### 标准模式功能

| 功能模块 | 说明 |
|---------|------|
| 🎬 视频播放 | ExoPlayer（默认）+ VLC（备用）双引擎，支持 4K/8K/HDR/杜比视界 |
| 🎵 音频播放 | 支持无损格式（FLAC/WAV/ALAC），歌词/专辑封面/播放列表 |
| 🖼️ 图片查看 | JPEG/PNG/WebP/HEIC 等格式 |
| 🏡 媒体库 | 电影/电视剧/音乐库，TMDB 自动刮削元数据 |
| 💬 弹幕 | B 站风格 XML 弹幕，支持自定义样式与显示区域 |
| 🕛 历史记录 | 音视频播放历史，断点续播 |
| 🔍 搜索 | 按标题/文件名模糊搜索本地媒体库 |
| 🎚️ 多轨道 | 音轨/视频轨/字幕轨切换 |
| 🌐 网络协议 | SMB / FTP / WebDAV / NFS / HTTP / 本地 |
| ⚙️ 设置 | 播放器内核、字幕外观、音频直通、解码模式等 |

### 老人模式（适老化）功能

| 功能 | 说明 |
|------|------|
| 固定栏位首页 | 大图卡片网格，4/6/8/12/15/16/20/24 个栏位可选，默认 5x3=15 |
| 全部应用入口 | 首页顶部固定按钮，点击进入应用列表页，可启动任意已安装 App |
| 零文字依赖 | 视觉优先，不依赖文字识别 |
| 一键播放 | 点击即播，自动续播无弹窗 |
| 简化控制栏 | 仅保留 ◀◀ ▶/❚❚ ▶▶ 三个大按钮 |
| 空栏位自助配置 | 点击空栏位可选择 App 添加，无需 Web 配置 |
| 应用卸载自动清理 | App 被卸载后，对应栏位自动重置为空 |
| Web 远程配置 | 局域网内通过浏览器管理栏位、连接、视频排序 |
| Launcher 模式 | 可设为 TV 默认桌面，开机直达 |
| 模式切换保护 | 长按下键 5 秒 + PIN 码切换到标准模式 |

## 1.4 双模式架构

项目支持两种运行模式，通过 `SettingsRepository.appMode` 控制：

```
┌─────────────────────────────────────────┐
│              MzDKPlayer App              │
│                                         │
│  ┌─────────────┐   ┌─────────────────┐  │
│  │  老人模式    │   │   标准模式       │  │
│  │  (默认)      │   │  (完整功能)      │  │
│  │             │   │                 │  │
│  │ · 固定栏位   │   │ · 侧边栏导航    │  │
│  │ · 一键播放   │   │ · TMDB 刮削     │  │
│  │ · 无文字依赖 │   │ · 弹幕/多协议   │  │
│  │ · 自动续播   │   │ · 全部设置      │  │
│  └─────────────┘   └─────────────────┘  │
│                                         │
│  切换: 长按下键5秒 → PIN → 标准模式      │
└─────────────────────────────────────────┘
```

模式切换由 [ModeManager.kt](../app/src/main/java/org/mz/mzdkplayer/data/repository/ModeManager.kt) 统一管理，导航起点在 [MzDKPlayerAPP.kt:175](../app/src/main/java/org/mz/mzdkplayer/ui/MzDKPlayerAPP.kt#L175)：

```kotlin
startDestination = if (ModeManager.isElderMode) "ElderHomePage" else "MainPage"
```

## 1.5 目标用户与硬件

### 目标用户

1. **家庭影音用户** - 拥有 NAS，希望在 TV 上播放局域网影视
2. **不识字老年人** - 通过老人模式，家属配置后一键观看
3. **影视爱好者** - 需要 TMDB 刮削、弹幕、多轨道等专业功能

### 硬件要求

| 等级 | 芯片 | 内存 | 系统 |
|------|------|------|------|
| 推荐 | Amlogic S928X-J | 4GB+ | Android TV 11+ |
| 一般 | MT9653 或同等 | 2GB | Android TV 7+ |
| 最低 | 晶晨 S905L 或同等 | 1GB | Android TV 7+（minSdk 23） |
