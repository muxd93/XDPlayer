# 07 - 开发环境搭建

## 7.1 环境要求

### 必需软件

| 软件 | 版本要求 | 说明 |
|------|---------|------|
| Android Studio | 最新稳定版 | 推荐 Hedgehog 或更新 |
| JDK | 21 | 项目使用 Java 21 编译 |
| Android SDK | compileSdk 37 | 需安装 SDK 36+ |
| Kotlin | 2.4.0 | 由 Gradle 自动管理 |
| Gradle | 由 wrapper 管理 | 使用 `./gradlew` |

### 硬件要求

- **开发机**：8GB+ RAM，SSD 存储
- **测试设备**：Android TV 设备（支持 ADB 调试）或 Android TV 模拟器

## 7.3 配置 TMDB API Key

TMDB 元数据刮削功能需要 API Key。**不配置也能编译**，但电影/电视剧详情页无法获取数据。

1. 注册 [TMDB](https://www.themoviedb.org/) 账号
2. 在设置中申请 API Key
3. 在项目根目录创建/编辑 `local.properties`：

```properties
TMDB_API_KEY=your_api_key_here
```

> `local.properties` 已在 `.gitignore` 中，不会提交。

## 7.4 构建配置

### 默认构建

```bash
# Debug 构建
./gradlew assembleDebug

# Release 构建
./gradlew assembleRelease
```

### ABI 拆分构建

默认构建 `armeabi-v7a` + `arm64-v8a` 两个分包 + 通用包：

```bash
# 只构建 arm64（包体积更小）
./gradlew assembleRelease -PtargetAbi=arm64

# 只构建 arm32
./gradlew assembleRelease -PtargetAbi=arm32

# 只构建 x86（模拟器）
./gradlew assembleRelease -PtargetAbi=x86
```

### 自定义版本号

```bash
./gradlew assembleRelease -PversionCode=98 -PversionName=1.16.0
```

## 7.5 运行与调试

### 连接设备

```bash
# 确认设备已连接
adb devices

# 通过网络连接 TV（需先在 TV 上开启网络调试）
adb connect <TV_IP>:5555
```

### 安装运行

```bash
# 安装 Debug 包
adb install -r app/build/outputs/apk/debug/app-debug.apk

# 或直接通过 Android Studio Run
```

### 查看日志

```bash
# 过滤应用日志
adb logcat | grep -E "MzDKPlayer|MainActivity|VideoPlayer|MediaInfo"

# 查看播放器错误
adb logcat | grep -E "MzExoPlayer|MzVlcPlayer|PlaybackException"
```

## 7.9 测试设备推荐

### 模拟器

- Android TV 模拟器（Android Studio AVD）
- 系统镜像：Android TV 11+ (API 30+)

### 真机

- 任何支持 ADB 调试的 Android TV 盒子
- 推荐芯片：Amlogic S905 系列或更高

### ADB 调试开启

1. TV 设置 → 关于 → 连续点击"版本号"7 次（开启开发者选项）
2. 开发者选项 → USB 调试（或网络调试）
3. 网络调试：`adb connect <TV_IP>:5555`
